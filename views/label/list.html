捐赠版功能

演示包含目前捐赠版的特有功能，且会持续更新。

文件搜索

模糊搜索

支持按照文件名模糊搜索，且支持配置是否搜索忽略大小写

搜索方式

支持搜索当前文件夹、当前文件夹及子文件夹、全局搜索三种模式

权限控制

支持配置仅管理员搜索还是匿名用户也可搜索

下载限制

限速下载

限速下载文件单链接的下载速度限制（仅支持本地存储、阿里云、腾讯云、FTP、SFTP、WebDav）

限制多线程下载

支持限制文件是否可多线程下载（仅本地存储支持）

权限控制

细致权限

支持控制管理员和匿名用户对文件操作、直链、忽略密码、忽略隐藏等功能进行权限配置。

安全可靠

采用前后端双重校验，避免被绕过权限判断。

性能优先

采用缓存机制，避免权限校验带来的性能损耗，给你畅快的体验.

统计分析

热点文件

通过统计热点文件，找出哪些文件被频繁下载。

热点 IP

通过统计热点 IP，找出哪些 IP 频繁引用了直链。

热点来源

通过统计热点来源，找出哪些网站引用了您的直链，并结合 Referer 防盗链，保护您的站点安全。

打包下载

打包下载

支持多选文件后打包下载，且可指定打包文件名

速度优先

不进行服务器中转，而是采用现代浏览器技术进行客户端打包，不会受限于服务器带宽。（仅支持 Chrome 浏览器）

WebDAV

动态开启

可动态启用或禁用 webdav，可以根据实际需求进行调整。

权限控制

支持使用管理员账号密码访问，会使用管理员权限查看文件和文件夹，也可以为游客权限的账号密码，不配置账号密码，则无需密码可访问 webdav.

兼容下载

支持中转下载也支持重定向下载，可以自由选择下载方式。
